# demand-weaver
Demand Weaver is a simple, mock end-to-end data project designed to demonstrate how product revenue trends can be analysed alongside external market signals. It combines sample e-commerce data with external sources to build a simple pipeline and dashboard that surface demand patterns.

## Project Structure

- `data/` → raw and processed data  
- `demand_weaver/` → core logic and pipeline  
- `streamlit_app/` → dashboard app
